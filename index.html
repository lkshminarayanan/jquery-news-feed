jQuery.Newsfeed


A jQuery plugin for displaying newsfeeds in web pages.


Download Plugin View it in Github

Using the Plugin

The plugin can be easily integrated to any web page. 

//1. include the plugin file.
<script src="jquery.newsFeed.js"></script>

//2. Load a newsContent in JSON format.
var newsContent = {
	"News 1":"Link 1",
	"News 2":"Link 2"
};

//3. call the plugin!
$('#divID').newsFeed({
	newsData: newsContent
});

Following options are available to modify the outcome of the plugin.

Option Description Values Default
type The id of the Newsfeed model to be rendered(Refer examples). 1/2/3 3
timeInterval Time to wait before changing to next news item(ms). >0 5000
targetBlank Specifies whether to open the link in a new tab. true/false true

The FadeIn/FadeOut : type 3
