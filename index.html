jQuery.Newsfeed


A jQuery plugin for displaying newsfeeds in web pages.


Download Plugin

Using the Plugin

The plugin can be easily integrated with any web page in simple steps. 

//1. include the file.
<script src="jquery.newsFeed.js"></script>

//2. create a newsContent in JSON format.
var newsContent = {
	"News 1":"Link 1",
	"News 2":"Link 2"
};

//3. call the plugin!
$('#divID').newsFeed({
	newsData: newsContent
});

Following options are available to tweak the effects.

Option Description Values
type The id of the Newsfeed model to be rendered(Check Examples). 1/2/3 (default 3)
timeInterval Time to wait before changing to next news item. >0
targetBlank Specifies whether to open the link in a new tab. true/false
